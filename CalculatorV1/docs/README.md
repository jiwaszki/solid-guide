### Some useful commands
```
git status

git remote add upstream https://github.com/jiwaszki/solid-guide
git remote -v

git fetch --all -p

git pull upstream main

git log --graph --branches --oneline --decorate

git checkout -b [your_name]/[what]/[exactly]

git stash save wip
git stash list
git stash apply 0

git checkout -
```
